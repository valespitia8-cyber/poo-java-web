Clases y Objetos en Java

Ejemplo básico de clase y objeto.

Ejemplo

class Persona {
    String nombre;
    int edad;

    void saludar(){
        System.out.println("Hola, soy " + nombre);
    }
}

public class Main {
    public static void main(String[] args){
        Persona p = new Persona();
        p.nombre = "Ana";
        p.edad = 20;
        p.saludar();
    }
}
