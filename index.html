Programación Orientada a Objetos en Java

Sitio educativo con definiciones, ejemplos y diagramas de POO en Java.

Estructuras de Control

Ejemplo de if-else:

int x = 10;
if(x > 5){
    System.out.println("x es mayor que 5");
} else {
    System.out.println("x es menor o igual a 5");
}

Ejemplo de for:

for(int i = 0; i < 5; i++){
    System.out.println("Número: " + i);
}

Clases y Objetos

Ejemplo básico de clase:

class Persona {
    String nombre;
    int edad;

    void saludar(){
        System.out.println("Hola, soy " + nombre);
    }
}

public class Main {
    public static void main(String[] args){
        Persona p = new Persona();
        p.nombre = "Ana";
        p.edad = 20;
        p.saludar();
    }
}

Pilares de la POO

Encapsulamiento

class Cuenta {
    private double saldo;
    public void depositar(double monto){ saldo += monto; }
    public double getSaldo(){ return saldo; }
}

Herencia

class Animal {
    void sonido(){ System.out.println("Sonido genérico"); }
}
class Perro extends Animal {
    void sonido(){ System.out.println("Guau guau"); }
}

Polimorfismo

Animal a = new Perro();
a.sonido(); // Imprime "Guau guau"

Abstracción

abstract class Figura {
    abstract double area();
}
class Circulo extends Figura {
    double radio;
    Circulo(double r){ radio = r; }
    double area(){ return Math.PI * radio * radio; }
}
