Sección 1: Estructuras de Control

En esta sección se explican los bucles más comunes en Java: while, do-while y for.

Bucle while

Definición: El bucle while ejecuta repetidamente un bloque mientras la condición sea verdadera. Se usa cuando no sabemos cuántas iteraciones serán necesarias de antemano.

Sintaxis:

while (condición) {
    // código a repetir
}

Ejemplo práctico (contar de 1 a 10):

int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;
}

Diagrama de flujo (while)

Bucle do-while

Definición: Similar a while, pero garantiza que el bloque se ejecute al menos una vez porque la condición se evalúa al final.

Sintaxis:

do {
    // código a ejecutar
} while (condición);

Ejemplo práctico (menú simple):

import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int opcion;
do {
    System.out.println("1) Saludar\n2) Sumar\n3) Salir");
    opcion = sc.nextInt();
    switch(opcion){
        case 1: System.out.println("¡Hola!"); break;
        case 2: System.out.println("2 + 2 = " + (2+2)); break;
    }
} while(opcion != 3);
sc.close();

Diagrama de flujo (do-while)

Bucle for

Definición: Ideal cuando conoce el número de repeticiones. Contiene inicialización, condición e incremento.

Sintaxis:

for (inicialización; condición; incremento) {
    // cuerpo
}

Ejemplo práctico (recorrer un arreglo):

String[] nombres = {"Ana","Luis","María"};
for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
}

Sección 2: Clases, Objetos y Métodos

Modelamos el mundo real con clases (plantillas) y objetos (instancias).

Clases y Objetos

Analogía: La clase es el molde; los objetos son las galletas. La clase define atributos y comportamientos.

Ejemplo práctico (Clase Coche):

class Coche {
    String color;
    String marca;
    int velocidad;

    Coche(String color, String marca, int velocidad){
        this.color = color;
        this.marca = marca;
        this.velocidad = velocidad;
    }

    void acelerar(int aumento){ velocidad += aumento; }
    void mostrar(){ System.out.println(marca + " " + color + " -> " + velocidad + " km/h"); }
}

Coche c1 = new Coche("rojo", "Toyota", 0);
Coche c2 = new Coche("azul", "Honda", 30);
c1.acelerar(20);
c1.mostrar();

Métodos y Overloading

Métodos: Son funciones dentro de la clase que describen comportamientos. Overloading: mismo nombre, diferentes parámetros.

Ejemplo (Calculadora):

class Calculadora {
    int sumar(int a, int b){ return a + b; }
    double sumar(double a, double b){ return a + b; } // sobrecarga
}

Uso: La JVM elige el método según los parámetros.

Modificador static

Explicación: Pertenece a la clase, no a la instancia. Útil para contadores y métodos utilitarios.
class EjemploStatic {
    static int contadorDeInstancias = 0;
    EjemploStatic(){ contadorDeInstancias++; }
    static int obtenerContador(){ return contadorDeInstancias; }
}

Clases Locales (Local Classes)

Definición: Clases definidas dentro de un método. Su alcance es local al bloque donde están definidas.
void metodoEjemplo() {
    class Mensajero {
        void enviar(String m){ System.out.println("Enviar: " + m); }
    }
    Mensajero ms = new Mensajero();
    ms.enviar("Hola desde clase local");
}

Sección 3: Pilares y Conceptos Avanzados

Principios para un diseño robusto y modular.

Encapsulamiento

Definición: Ocultar el estado interno y exponer operaciones seguras (métodos).
class CuentaBancaria {
    private double saldo;

    public void depositar(double monto){ if(monto > 0) saldo += monto; }
    public boolean retirar(double monto){
        if(monto <= saldo){ saldo -= monto; return true; }
        return false;
    }
    public double getSaldo(){ return saldo; }
}

Composición (has-a)

Definición: Un objeto está formado por otros objetos.

Ejemplo práctico:

class Motor { int potencia; Motor(int p){ potencia = p; } }
class Coche {
    Motor motor;
    Coche(Motor m){ this.motor = m; }
}

Delegación

Definición: Un objeto delega tareas a otro que tiene la responsabilidad especializada.
class Impresora {
    void imprimir(String texto){ System.out.println("Imprimiendo: " + texto); }
}
class Gerente {
    Impresora impresora = new Impresora();
    void imprimirReporte(){ impresora.imprimir("Reporte ejecutivo"); }
}

Clases Envoltorio (Wrapper Classes)

Definición: Tipos de objeto que envuelven tipos primitivos (Integer, Double...). Útiles en colecciones y para métodos genéricos.
// Autoboxing y unboxing
int n = 5;
Integer N = n; // autoboxing
int m = N;     // unboxing

// Usando en colecciones
java.util.ArrayList<Integer> lista = new java.util.ArrayList<>();
lista.add(10); // autoboxing
