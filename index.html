Estructuras de Control

if-else: Es una estructura condicional que permite ejecutar un bloque de código si la condición es verdadera y otro bloque si es falsa. 
int x = 10;
if(x > 5){
    System.out.println("x es mayor que 5");
} else {
    System.out.println("x es menor o igual a 5");
}
for: Es un ciclo que repite un bloque de código un número determinado de veces, muy útil para recorrer rangos o colecciones. 
for(int i = 0; i < 5; i++){
    System.out.println("Número: " + i);
}

Clases y Objetos

Clase: Es una plantilla que define atributos y métodos. Objeto: Es una instancia de una clase, con valores propios. 
class Persona {
    String nombre;
    int edad;

    void saludar(){
        System.out.println("Hola, soy " + nombre);
    }
}

public class Main {
    public static void main(String[] args){
        Persona p = new Persona();
        p.nombre = "Ana";
        p.edad = 20;
        p.saludar();
    }
}

Pilares de la POO

Encapsulamiento: Oculta los datos internos de un objeto y permite acceder a ellos solo mediante métodos públicos.
class Cuenta {
    private double saldo;
    public void depositar(double monto){ saldo += monto; }
    public double getSaldo(){ return saldo; }
}
Herencia: Permite crear nuevas clases basadas en clases existentes, reutilizando código.
class Animal {
    void sonido(){ System.out.println("Sonido genérico"); }
}
class Perro extends Animal {
    void sonido(){ System.out.println("Guau guau"); }
}
Polimorfismo: Permite que un objeto adopte diferentes formas, usando el mismo método con comportamientos distintos.
Animal a = new Perro();
a.sonido(); // Imprime "Guau guau"
Abstracción: Permite definir métodos sin implementación en clases abstractas, que serán desarrollados en clases hijas.
abstract class Figura {
    abstract double area();
}
class Circulo extends Figura {
    double radio;
    Circulo(double r){ radio = r; }
    double area(){ return Math.PI * radio * radio; }
}
